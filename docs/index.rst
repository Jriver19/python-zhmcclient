
zhmcclient - A Pure python client library for the z Systems HMC Web Services API
********************************************************************************

The **zhmcclient** PyPI package provides a client library for the Web Services
API of the z Systems Hardware Management Console, written in pure Python.
It supports Python 2 and Python 3.

This documentation describes the API of the **zhmcclient** PyPI package.

Its general web site is: http://zhmcclient.readthedocs.org.
